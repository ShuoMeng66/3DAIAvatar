## 使用建议

- 作为 ElderTalk 老人陪伴应用的默认数字人形象源图
- 适用于 MuseTalk、SadTalker、Wav2Lip 等音频驱动人脸动画模型
- 建议选择正面、光线均匀、面部清晰无遮挡的肖像照片以获得最佳驱动效果
- 如需更好的效果，可使用真人照片替换为特定形象
